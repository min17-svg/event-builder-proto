_S03 트래픽 개요 (기간 필터 + 꺾은선 차트) ›
EVENT_SUMMARY_03
트래픽 개요 (기간 필터 + 꺾은선 차트)
조회 기간 선택 드롭다운 + 꺾은선 차트 + 범례
이벤트 기간 중 날짜별 DT·PV·UV를 꺾은선 차트로 표시한다.
  • 기간 필터 4종 (기본: 전체):
    • 전체 — 이벤트 전체 기간 (시작일 ~ 종료일). 기본 선택값.
    • 7일 — 오늘 기준 최근 7일 (이벤트 기간과 교집합)
    • 한달 — 오늘 기준 최근 30일 (이벤트 기간과 교집합)
    • 사용자 지정 — 날짜 피커에서 직접 선택. 선택 가능 범위는 이벤트 기간(시작일 ~ 종료일)으로 제한
  • DT(체류시간) · PV(페이지뷰) · UV(순방문자수) — 세 지표가 엇갈리면 문제의 단서
연동 고려사항 기간 필터 변경 시 차트 · 이벤트 유입 현황 테이블 · 미션형 이벤트 지표 테이블 세 곳을 동시에 갱신한다.
Alert (사용자 지정 기간 선택시) 시작일 > 종료일 상태로 "적용" 클릭 → "시작일이 종료일보다 늦습니다. 날짜를 다시 확인해 주세요." [확인]
_S04 이벤트 지표 (탭 + 테이블 + 원본 데이터 (CSV)) ›
EVENT_SUMMARY_04
이벤트 지표 (탭 + 테이블 + 원본 데이터 (CSV))
조회 섹션 헤더(타이틀 + Outline 버튼 "원본 데이터 (CSV)") + 탭 바 + 데이터 테이블
"이벤트 지표" 타이틀과 원본 데이터 (CSV) 버튼이 같은 행에 배치(타이틀 좌측, 버튼 우측). 탭으로 두 가지 뷰를 전환한다.
  • 탭 1 — 이벤트 유입 현황 (이벤트 노출·도달 관점)
    컬럼: 일자 / UV / PV / DT / 평균 접속 횟수
    상단에 기간 전체 합계 행(전체) 고정 노출, 날짜 내림차순 정렬
  • 탭 2 — 미션형 이벤트 지표 (이벤트 참여·행동 관점)
    컬럼: 일자 / 타입 / 달성자 수(유니크) / 달성 횟수 / 보상 획득 유저 수(유니크) / 보상 지급 횟수
    날짜별 미션 타입 단위로 행 분리, 상단에 전체 집계 행 고정 노출
원본 데이터 (CSV) 버튼 동작 규칙
화면 필터(기간·유저군 등)와 무관하게 이벤트 전체 기간 데이터를 다운로드한다.
이벤트가 진행 중이면 시작일 ~ 오늘까지의 데이터를 다운로드한다.
CSV 파일은 monolake에서 볼 수 있는 파일로 제공된다.

버튼 툴팁
첫 줄 (항상 표시): "화면 필터와 무관하게 이벤트 전체 기간 데이터를 받습니다."
둘째 줄 (진행 중일 때만 추가): "이벤트가 진행 중이면 시작일 ~ 오늘까지 데이터를 받습니다."

  • 14일 — 오늘 기준 최근 14일, 이벤트 기간 내로 클램핑
  • 사용자 지정 — 날짜 입력행 노출, 시작일·종료일 입력 후 [적용] 클릭 시 반영. 선택 범위는 이벤트 기간 내로 제한
  • 현재 적용 기간은 YY.MM.DD ~ YY.MM.DD · N일 형식으로 우측에 항시 표시
연동 지표:
  • 유저군별 참여율 KPI / 참여율 테이블 — 선택 기간 데이터로 스케일

  • 일별 참여·미참여 유저 수 / 일별 리텐션 테이블 — 기간 내 날짜 행만 표시

EVENT_DETAIL_04
유저군 × 레벨군별 참여율 테이블
조회 계층형 데이터 테이블 (유저군 › 레벨군) · 인라인 바 차트
유저군(기존/복귀/신규) × 레벨군(저/중/고레벨) 교차 조합별 참여율 테이블.
  • 컬럼: 유저군 / 레벨군 / 참여 유저수 / 미참여 / 전체 / 참여율(%) / 참여율 바 차트
  • 레벨군: 저레벨(Lv.1~40) / 중레벨(Lv.41~70) / 고레벨(Lv.71~90)
  • 유저군 합계 행: 배경색 + 굵은 텍스트로 강조
  • 바 차트를 숫자 옆에 병행 표시 → 레벨군 간 높낮이를 시각적으로 즉시 식별
레벨군 설정은 게임이 처음 이벤트 빌더에 연동될 시 등록되며 변경은 이벤트 빌더 담당자에게 문의가 필요합니다.

_D07 참여·미참여 리텐션 비교 (D+n 탭) ›
EVENT_DETAIL_07
참여·미참여 리텐션 비교 (D+n 탭)
조회 D+n 탭 바 + 유저군 × 레벨군 3×3 그리드 (참여/미참여 바 차트 + 차이)
이벤트 참여·미참여 유저의 리텐션을 유저군 × 레벨군별로 비교. 각 셀: 참여 리텐션 바 · 미참여 리텐션 바 · 차이(%p).
  • 탭 구성 규칙 (기간 필터 연동): [1, 7, 14, 21, 28] 중 선택 기간 일수 이하의 값만 표시. 데이터 최대 범위 D+28.
    • 7일 이하 → D+1, D+7
    • 8~14일 → D+1, D+7
    • 15~29일 → D+1, D+7, D+14
    • 30일 이상 → D+1, D+7, D+14, D+21, D+28
  • D+1 — 단기 점착도 (다음 날 즉각 재방문 유도했는가)
  • D+7 — 중기 리텐션 (일주일 단위 습관 형성에 기여했는가)
  • D+14 — 장기 습관화 (이벤트 종료 후에도 효과가 지속되는가)
  • D+21 — 3주차까지 효과가 이어지는가. 이벤트 후 중장기 잔류 여부
  • D+28 — 4주(한 달) 시점 리텐션. 이벤트 종료 후 습관 정착 여부
  • 빨간 강조(역전 현상): 참여 리텐션 < 미참여 리텐션 — 이벤트가 오히려 이탈을 유도했을 가능성. 주로 미션 난이도 과도하거나 보상 실망 시 발생
